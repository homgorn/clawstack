# Weekly Intel

## Goal

Produce a short weekly report with:

- security issues;
- OpenClaw changes;
- new or changed low-cost providers;
- config changes worth considering.

## Suggested cadence

- every Sunday;
- isolated session;
- send alerts immediately if a security issue is discovered.

## Output format

Create a concise report with:

1. urgent security notes
2. important platform changes
3. pricing or provider shifts
4. recommended config updates
