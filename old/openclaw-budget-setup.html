🦞Что такое OpenClaw

OpenClaw — open-source AI агент-рантайм (MIT лицензия), бывший Clawdbot / Moltbot. Создан Питером Штейнбергером в ноябре 2025. За 2 недели набрал 200 000+ GitHub звёзд — один из самых быстрорастущих репозиториев в истории.

Суть: Локальный Gateway на Node.js запускается у тебя и принимает сообщения из WhatsApp/Telegram/Discord/Slack → передаёт в LLM → LLM выполняет инструменты (браузер, shell, файлы, email, API) → отвечает тебе. Всё локально. Модели — внешние API.
⚡ Что умеет
  • Email, календарь, файлы
  • Веб-браузинг, shell-команды
  • Мультиагентная маршрутизация
  • Heartbeat-планировщик (cron)
  • 50+ платформ: WA, TG, Discord…
  • Self-modifying skills
💡 Ключевые файлы
  • openclaw.json — главный конфиг
  • SOUL.md — личность агента
  • AGENTS.md — команда агентов
  • USER.md — данные о тебе
  • HEARTBEAT.md — фоновые задачи
  • TOOLS.md — разрешённые инструменты

🏗️Архитектура умной маршрутизации

Ключевая идея — не гонять всё через одну дорогую модель. Распределяем задачи по сложности и цене.

Ты (Telegram / WhatsApp / Discord) │ ▼ OpenClaw Gateway (:18789) — Node.js, локальный, Lane Queue (без race conditions) │ ▼ iblai-router / ClawRouter (:8402) — 14-мерный скорер сложности, <1ms │ ├── score < 0.0 → LIGHT → OpenRouter :free (Llama 70B / Gemini Flash) │ Цена: $0.00 | Использовать: приветствия, поиск файлов, погода │ ├── score 0.0–0.35 → MEDIUM → DeepSeek V3.2 / Gemini 2.5 Flash │ Цена: ~$0.50/M | Использовать: анализ, summary, draft │ └── score > 0.35 → HEAVY → Claude Sonnet / Haiku (через API или подписку) Цена: $1–3/M | Использовать: код, сложная логика, агентные задачи Агентная команда (параллельные воркеры): ├── agent:coordinator → Claude Haiku / Sonnet (менеджер задач) ├── agent:researcher → Kimi K2.5 / DeepSeek R1:free (поиск инфы) ├── agent:coder → DeepSeek V3.2 (код, дешевле x50 Opus) ├── agent:monitor → GPT-OSS-20B:free (heartbeat, мониторинг) └── agent:writer → Gemini Flash (тексты, summary) Heartbeat каждые 30 мин → Gemini 2.5 Flash-Lite ($0.50/M) = почти $0/мес
Главное правило: Дорогая модель — только для сложных задач. Heartbeats, мониторинг и простые запросы — на бесплатных или Flash-Lite. Это снижает расходы на 50–80% без потери качества.

⚙️iblai-router — Умный роутер задач (Open Source)

Open-source прокси, который анализирует каждый запрос по 14 измерениям и выбирает нужную модель. Работает между OpenClaw и Anthropic API.

Ссылка: github.com/iblai/iblai-openclaw-router — ~250 строк кода, zero deps, горячая перезагрузка конфига

Как работает скорер

Запрос пользователя → iblai-router (:8402) → 14-мерный анализ (<1ms) 14 измерений: 1. Количество токенов 8. Сложность вопроса 2. Наличие кода 9. Императивные глаголы 3. Маркеры рассуждений 10. Ограничения/constraints 4. Технические термины 11. Формат вывода (JSON, таблица) 5. Творческие маркеры 12. Специфика домена 6. Простые индикаторы 13. Агентные задачи 7. Многошаговые паттерны 14. Relay индикаторы Результат: score < 0.0 → LIGHT → Claude Haiku ($1/$5 per 1M) 0.0 – 0.35 → MEDIUM → Claude Sonnet ($3/$15 per 1M) score > 0.35 → HEAVY → Claude Opus ($15/$75 per 1M) Overrides: • 2+ ключевых слова "reasoning/analyze" → HEAVY (0.95 confidence) • >50K токенов в контексте → HEAVY • Низкая уверенность → MEDIUM (безопасный дефолт)

Установка

# Клонируем роутер
git clone https://github.com/iblai/iblai-openclaw-router
cd iblai-openclaw-router

# Запускаем прокси (Node.js, zero dependencies)
node server.js

# Проверяем здоровье
curl http://127.0.0.1:8402/health

# Тестируем роутинг
curl http://127.0.0.1:8402/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: test" \
  -H "anthropic-version: 2023-06-01" \
  -d '{"model":"auto","max_tokens":50,"messages":[{"role":"user","content":"Say hi"}]}' | jq .model

# Если вернёт "claude-haiku-..." — роутер работает!

👥AGENTS.md — Описание команды

Создай файл ~/.openclaw/workspace/AGENTS.md:

# Agent Team Configuration

## Coordinator (main agent — это ты)
Твоя роль: координировать задачи и делегировать правильным агентам.

**Правила делегирования:**
- Веб-поиск, исследования → researcher (бесплатная модель)
- Написание/рефакторинг кода → coder (DeepSeek V3.2)
- Тексты, письма, summary → writer (Gemini Flash)
- Мониторинг систем → monitor (GPT-OSS бесплатно)
- Сложные рассуждения → сам через /model sonnet

## Как вызывать агентов:
Используй sessions_send() для передачи задач:
- "Отправь задачу researcher: найди информацию о..."
- "Передай coder: напиши функцию для..."
- "Попроси writer: составь email..."

## Правило экономии:
Сначала попробуй сделать сам (Haiku).
Только если задача явно требует — эскалируй к дорогой модели.
Используй /model sonnet только для действительно сложного.

👁️SOUL.md + HEARTBEAT.md

~/.openclaw/workspace/SOUL.md

# Core Identity
Я — умный координатор задач. Работаю эффективно, без лишних слов.

# Принципы работы
- Сначала думаю, потом действую
- Простые задачи делаю сам, сложные — делегирую
- Слежу за расходами на токены как за своими деньгами
- Предупреждаю о потенциально дорогих операциях

# Стиль общения
- Кратко и по делу. Нет воды.
- Если ответ в одно предложение — так и отвечаю
- Технические термины использую точно
- Честно говорю, когда не знаю

# Правила экономии
- Heartbeat: только GPT-OSS или Gemini Flash-Lite
- Sub-agents: DeepSeek V3.2 для кода, Llama :free для research
- Главный агент: Haiku — достаточно для 80% задач
- Sonnet/Opus — только если задача явно требует

# Запрещено
- Никогда не запускаю бесконечные retry loops
- Не пишу на существующие файлы без подтверждения
- Не превышаю maxConcurrent=4 задачи

~/.openclaw/workspace/HEARTBEAT.md

# Heartbeat Tasks (каждые 30 минут, модель: Gemini Flash-Lite)

## Приоритет HIGH
- Проверить новые важные письма (subject содержит: urgent, важно, deadline)
- Проверить уведомления из CI/CD систем

## Приоритет MEDIUM  
- Краткая сводка задач на день (только утром, до 10:00)
- Проверить не забытые задачи из TASKS.md

## Правила heartbeat
- Если ничего срочного → ответить HEARTBEAT_OK, не беспокоить
- Не отправлять больше 1 сообщения за heartbeat
- Использовать максимально дешёвую модель
