# Configs

Starter configs for the public repo.

- [openclaw-budget.json](openclaw-budget.json): paid-but-efficient baseline
- [openclaw-free.json](openclaw-free.json): free-first baseline
- [openclaw.schema.json](openclaw.schema.json): JSON schema for editor hints

Treat both as starting points, not copy-paste truth for every environment.

## Validation

Optional repo lint for these starter configs:

python ../scripts/check_configs.py
